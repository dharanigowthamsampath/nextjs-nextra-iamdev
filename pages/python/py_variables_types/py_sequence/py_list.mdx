# Lists in Python

A **list** in Python is an **ordered, mutable collection** used to store multiple items in a single variable. Lists are extremely flexible, capable of storing any data type—including other lists.

They are one of the most commonly used data structures in Python due to their versatility and dynamic nature.

---

## Key Characteristics of Lists

- **Mutable**: You can add, remove, or modify items after creation.
- **Ordered**: The order in which items are inserted is preserved.
- **Heterogeneous**: Lists can hold items of **different types**.
- **Indexed**: Items can be accessed by position using square brackets `[]`.

---

## Creating Lists

```python filename="list_create.py" {1, 2}
empty_list = []
numbers = [1, 2, 3, 4, 5]
```

### Explanation:

- `[]` creates an empty list.
- `[1, 2, 3, 4, 5]` creates a list of integers.

---

## Example 1: Accessing Elements with Indexing

```python filename="list_indexing.py" {2, 3, 4}
fruits = ["apple", "banana", "cherry"]
print(fruits[0])     # First element
print(fruits[-1])    # Last element
```

```python filename="output.txt"
apple
cherry
```

### Explanation:

- `fruits[0]` retrieves the first element.
- Negative indexing (`fruits[-1]`) accesses elements from the end.

---

## Example 2: Modifying Elements

```python filename="list_modify.py" {2}
fruits = ["apple", "banana", "cherry"]
fruits[1] = "blueberry"
print(fruits)
```

```python filename="output.txt"
['apple', 'blueberry', 'cherry']
```

### Explanation:

- Lists are **mutable**, so values can be updated by directly assigning to an index.

---

## Example 3: List Methods – `append()`, `insert()`, `extend()`

```python filename="list_methods.py" {3, 4, 5, 6}
colors = ["red", "green"]
colors.append("blue")                # Adds to end
colors.insert(1, "yellow")           # Inserts at index 1
colors.extend(["black", "white"])    # Adds multiple items
print(colors)
```

```python filename="output.txt"
['red', 'yellow', 'green', 'blue', 'black', 'white']
```

### Explanation:

- `append()` adds one item.
- `insert()` adds at a specific position.
- `extend()` merges another list (or any iterable).

---

## Example 4: Removing Elements – `remove()`, `pop()`, `del`

```python filename="list_remove.py" {2, 3, 4, 5}
nums = [10, 20, 30, 40, 50]
nums.remove(30)   # Removes value 30
nums.pop(1)       # Removes item at index 1
del nums[0]       # Deletes item at index 0
print(nums)
```

```python filename="output.txt"
[40, 50]
```

### Explanation:

- `remove(value)` deletes the first occurrence of the value.
- `pop(index)` removes and returns the element at the given index.
- `del` is a Python statement that deletes an item by index.

---

## Example 5: Slicing Lists

```python filename="list_slicing.py" {2, 3}
data = [0, 1, 2, 3, 4, 5, 6]
print(data[2:5])     # Slice from index 2 to 4
print(data[:3])      # First 3 elements
```

```python filename="output.txt"
[2, 3, 4]
[0, 1, 2]
```

### Explanation:

- `list[start:end]` returns a sublist from `start` to `end - 1`.
- Omitting `start` or `end` defaults to the beginning or end.

---

## Example 6: Iterating Through a List

```python filename="list_iteration.py" {2, 3}
animals = ["cat", "dog", "elephant"]
for animal in animals:
    print("Animal:", animal)
```

```python filename="output.txt"
Animal: cat
Animal: dog
Animal: elephant
```

### Explanation:

- You can use a `for` loop to access each element one by one.

---

## Example 7: Nested Lists (2D Lists)

```python filename="list_nested.py" {1, 2, 3}
matrix = [
    [1, 2, 3],
    [4, 5, 6]
]
print(matrix[1][2])   # Access row 2, column 3
```

```python filename="output.txt"
6
```

### Explanation:

- Nested lists are lists within lists.
- `matrix[1][2]` accesses the 2nd row’s 3rd element.

---

## Common List Methods

| Method      | Description                                |
| ----------- | ------------------------------------------ |
| `append()`  | Adds a single element to the end           |
| `extend()`  | Adds all elements from another iterable    |
| `insert()`  | Inserts element at specified position      |
| `remove()`  | Removes first matching value               |
| `pop()`     | Removes and returns element at given index |
| `clear()`   | Removes all items from the list            |
| `index()`   | Returns index of first matching value      |
| `count()`   | Returns number of occurrences of a value   |
| `sort()`    | Sorts the list in ascending order          |
| `reverse()` | Reverses the order of elements in place    |
| `copy()`    | Returns a shallow copy of the list         |

---

## Summary

- Lists are **ordered, mutable, and dynamic** containers.
- They support indexing, slicing, and nesting.
- Python provides powerful built-in methods for list operations.
- Understanding lists lays the groundwork for advanced data structures in Python.
